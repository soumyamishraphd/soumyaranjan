Intro

As an experienced computer science researcher with expertise in computer vision and machine learning, I am a dedicated Assistant Professor at KIIT University in Bhubaneswar, Odisha, India, With a PhD in Computer Science, from National Institute of Technology, I specialize in human action recognition, pattern recognition, image processing, and machine learning. I have gained hands-on experience in machine learning and OpenCV with Python and have published numerous articles in highly esteemed SCI and Scopus-indexed journals with exceptional impact factors. Additionally, I actively serve as a reviewer for several prestigious journals and conferences, further demonstrating my commitment to advancing the field of computer science. Overall, I am a highly skilled and motivated individual with a passion for using my expertise to make significant contributions to the field of computer science.

Skills

  • Programming Languages: Proficiency in programming languages such as Python, C++, or Java is crucial for implementing machine learning algorithms, working with frameworks, and developing computer vision applications.
  • Machine Learning Algorithms: Strong knowledge and understanding of machine learning algorithms, including supervised learning, unsupervised learning, and deep learning architectures (e.g., convolutional neural networks, recurrent neural networks).
  • Deep Learning Frameworks: Experience with deep learning frameworks such as TensorFlow, PyTorch, or Keras is essential for developing and training complex neural network models.
  • Computer Vision Techniques: Understanding of computer vision concepts and techniques, including image processing, feature extraction, object detection, segmentation, tracking, and recognition.
  • Neural Network Architectures: Familiarity with various neural network architectures for computer vision tasks, such as CNNs (Convolutional Neural Networks), RNNs (Recurrent Neural Networks), and GANs (Generative Adversarial Networks).
  • OpenCV: Proficiency in using OpenCV (Open Source Computer Vision Library) for image and video processing, manipulation, and analysis.
  • Data Preprocessing: Knowledge of data preprocessing techniques, including data cleaning, normalization, and feature engineering, to prepare data for machine learning models.
  • Statistical Analysis: Understanding of statistical concepts and methods for analyzing and interpreting data, including hypothesis testing, regression analysis, and probability theory.
  • Model Evaluation and Metrics: Familiarity with evaluation metrics for assessing the performance of machine learning models, such as accuracy, precision, recall, F1-score, and ROC curves.
  • Data Visualization: Proficiency in data visualization libraries like Matplotlib or Seaborn to effectively present and interpret the results of machine learning and computer vision experiments.
  • GPU Programming: Experience with GPU programming using frameworks like CUDA for accelerating computations in deep learning and computer vision tasks.
  • Version Control Systems: Knowledge of version control systems like Git for effective collaboration, code management, and tracking changes in machine learning projects.
  • Problem Solving and Debugging: Strong problem-solving skills and the ability to debug and troubleshoot issues that arise during the development and deployment of machine learning and computer vision applications.
  • Domain Knowledge: Understanding the specific domains where machine learning and computer vision are applied, such as autonomous vehicles, robotics, healthcare, security, or augmented reality.
